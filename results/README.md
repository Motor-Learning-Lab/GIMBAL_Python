# Results Directory

Test outputs, diagnostic plots, benchmarks, and generated figures.

## Structure

- **diagnostics/**: Diagnostic test outputs
  - `v0_2_1_divergence/`: v0.2.1 divergence diagnostics
    - `report.md`: Generated test report
    - `plots/`: Diagnostic plots (PNG files)

- **benchmarks/**: Performance benchmark results

- **figures/**: Generated figures for papers/documentation

## Git Policy

This directory is typically `.gitignore`d to avoid committing large output files.
Results can be regenerated by running the corresponding tests/scripts.

## Key Files

- `diagnostics/v0_2_1_divergence/report.md`: Main divergence test report
- `diagnostics/v0_2_1_divergence/plots/*.png`: Diagnostic visualizations
